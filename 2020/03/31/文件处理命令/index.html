Linux文件处理命令

  • 一键三连

命令格式与目录处理命令

命令格式： 命令 [-选项] [参数]

ls 命令

原意：list

命令所在路径： /bin/ls

执行权限： 所有用户

功能： 显示目录文件

语法 ： ls 选项[-ald] [文件或目录]

​ -a (all) 显示所有文件，包括隐藏文件

​ -l (long)详细信息显示

​ -d 查看目录属性

​ -h (human) 人性化显示

​ -i 查看id号

文件类型

’ - ‘ 表示权限

‘ d ’ （document）代表目录

‘ l ’ （link）代表软链接

ugo权限解释

例如 rw- r– r–

​ u g o

u所有者 g所属组 o其他

r读 w写 x可执行

文件与用户关系

所有者：一般为创建该文件的人（所有者只能有一个）

所属组：可以使用该文件的一群人（只能有一组）

其他：剩余的用户

目录处理指令

mkdir命令

原意：make directories

命令所在路径： /bin/mkdir

执行权限： 所有用户

功能： 创建新目录

​ -p递归创建（可以在本身不存在的目录下创建新的子目录【根目录和子目录一同被创建】）

语法 ：mkdir -p [目录名]

例如：mkdir /tmp/text

​ mkdir -p /tmp/text2/text3

rmdir命令

原意：remove emptydirectories

命令所在路径：/bin/rmdir

执行权限：所有用户

语法：rmdir [目录名]

功能：删除空目录

cd指令

原意：change directory

命令所在路径： shell 内置命令

执行权限：所有用户

语法：cd [目录]

功能：切换目录

pwd指令

原意：print working directory

命令所在路径：/bin/pwd

执行权限：所有用户

语法：pwd

功能：显示当前目录

cp命令

原意：copy

命令所在路径：/bin/cp

执行权限：所有用户

语法：cp -rp [原文件或目录] [目标目录]

​ -r 复制目录

​ -p 保留文件属性（时间属性不会发生改变）

功能：复制文件或目录，可以同时复制多个文件或目录

复制的同时可以改名

mv命令

原意：move

命令所在路径：/bin/mv

执行权限：所有用户

语法：mv [原文件或目录]

功能：剪切文件，改名

rm命令

原意：remove

命令所在路径：/bin/rm

执行权限：所有用户

语法：rm -rf [文件或目录]

​ -r 删除目录

​ -f 强制执行

功能：删除文件，目录

文件处理命令

touch命令

命令所在路径：/bin/touch

执行权限：所有用户

语法：touch [文件名]

功能：创建空文件

cat命令

命令所在路径：/bin/cat

执行权限：所有用户

语法：cat [文件名]

功能：显示文件内容

​ -n 显示行号

不适合浏览大文件

tac命令

命令所在路径：/usr/bin/tac

执行权限：所有用户

语法：tac [文件名]

功能：显示文件内的内容（反向列示）

more命令

命令所在路径：/bin/more

执行权限：所有用户

语法：more [文件名]

​ (空格)或f 翻页

​ (Enter) 换行

​ q或Q 退出

功能：分页显示文件内容 （适合大容量读，不可上翻）

less命令

命令所在路径：/usr/bin/less

执行权限：所有用户

语法：less [文件名]

功能：分页显示文件内容（可以向上翻页）

​ [pageup] 向上翻一页

​ 上箭头 向上翻一行

​ /[关键词] 自动搜索 按n继续往下找

head命令

命令所在路径：/usr/bin/head

执行权限：所有用户

语法：head [文件名]

功能：显示文件前面几行

​ -n 指定行数

tail命令

命令所在路径：/usr/bin/tail

执行权限：所有用户

语法：tail[文件名]

功能：显示文件后面几行

​ -n 指定行数

​ -f 动态显示文件末尾内容

链接命令

ln命令

原意：link

命令所在路径：/bin/ln

执行权限：所有用户

语法：ln -s [原文件] [目标文件]

​ -s 创建软链接

功能描述：生成链接文件

例子： ln -s /etc/issue /tmp/issue.soft 创建软链接

​ ln /etc/issue /tmp/issue.hard 创建硬链接

软链接与硬连接的特点与不同

软链接特征：类似于Windows中的快捷方式

硬连接特征：拷贝cp -p + 同步更新{硬连接不能跨分区，不能对目录使用}

​ （源文件丢失硬连接文件也能访问，于源文件id号相同）
